I'm a Computer Science student at IGDTUW who likes turning ideas into working systems — from a computer-vision research project at DRDO to a full-stack placement-prep tracker. I care about writing code that's clean and actually gets used, not just code that runs once in a notebook.

## 💼 Experience

### Research Intern — DRDO SAG (Scientific Analysis Group)
*4 weeks · Mentor: Ms. Mehak*

Investigated how background changes affect a vision-language model's (CLIP) predictions. Built a full pipeline: foreground/background separation with U²-Net, background similarity search with ResNet-18 + cosine similarity across ~8,700 images, and controlled background-swap experiments evaluated with CLIP ViT-B/32.
